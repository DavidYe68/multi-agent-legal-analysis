你需要根据被告人的人格特质，生成其在刑事庭审中的表达风格说明。

人格维度包括：
1. extraversion 外向性
2. neuroticism 神经质
3. openness 开放性
4. agreeableness 宜人性
5. conscientiousness 尽责性

请根据输入的人格等级，生成：
1. speech_style：说话风格
2. emotional_tendency：情绪倾向
3. cooperation_level：配合程度
4. detail_level：回答详略
5. risk_note：该人格可能带来的庭审表达风险

注意：
- 不要生成案件事实
- 不要判断有罪无罪
- 只描述表达风格

输出 JSON：
{
  "speech_style": "",
  "emotional_tendency": "",
  "cooperation_level": "",
  "detail_level": "",
  "risk_note": ""
}
