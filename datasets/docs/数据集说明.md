# 法律多智能体协商数据集 v2

## 数据集定位
本数据集不是裁判结果预测集，而是用于评估争点识别、证据映射、证据缺口发现、不确定性保持和关键事实敏感性。

## 数据规模
- 12个案例；
- 6组A/B最小事实对照；
- 覆盖帮助信息网络犯罪活动、盗窃、故意伤害、交通肇事、诈骗和证据可采性；
- 开发集8例，测试集4例。

## 构造方法
现有5个教学案例先经过事实、证据、主张分离和答案泄漏清理，再围绕核心法律争点构造A/B版本。每组只改变一个具有法律意义的关键变量。

## 金标准
每案标注事实证明状态、核心争点、证据方向与强度、证据局限、证据缺口、合理结论范围、多视角分歧与对照预期变化。

## 处理流程
材料整理 → 匿名化 → 事实/主张/证据分离 → 稳定编号 → 金标准标注 → A/B对照构造 → 泄漏检查 → 开发/测试划分。

## 说明
本版本为匿名化、要件驱动的合成教学数据，不对应单一真实自然人或具体案件，不用于真实司法裁判。数据由AI辅助初标，并由具有法学与人工智能交叉背景的项目成员复核。
